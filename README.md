Petclinic is a Spring Boot application built using Maven.
What I used in this application:
1-Maven
2-Hibernate
3-spring MVC
4-Bean Validation (Hibernate Validator)
5-Logback
6-Lombok
7-thymeleaf
8-bootsrtap
9-H2 DB
